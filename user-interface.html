    # WPF GUI components

    Two new windows were created using WPF to display the specifics of leave calculations - Leave Transactions, Leave Balance figures, and Leave Policy records.

    • the MapsUI window for detailed transaction and balance info on a per-person basis

    • the Admin window to manage leave policy configuration such as

      • Leave Groups (and association to Contracts)
      • Rate records to control the amount of earning and burning of leave for various activities
      • Legislative adjustment switches
      • Maximum amounts for Advanced and Anticipated leave
